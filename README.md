# LVT have KML view

QGIS Plugin to convert Shapefile (.shp) to KML/KMZ with customizable HTML popup for Google Earth Mobile.

## Features

- Dynamic field reading from any Shapefile
- Customizable Name label (combine 2 fields with separator)
- Rich HTML popup with field alias mapping
- Polygon styling: border color/width, fill color/opacity
- Conditional coloring with =, >, < operators
- Row highlight for special conditions in popup
- Save/Load configuration (JSON)
- KML and KMZ output
- Auto reproject to EPSG:4326
